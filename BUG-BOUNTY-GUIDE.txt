================================================
  4. BURPSUITE INTEGRATION
================================================

PREREQUISITES:
  - BurpSuite Professional (required for REST API)
  - BurpSuite REST API extension enabled
  - API running on http://127.0.0.1:1337

SETUP BURPSUITE:
  1. Start BurpSuite Professional
  2. Go to Extensions > BApp Store
  3. Install "REST API" extension
  4. Configure API key (optional)
  5. Note API URL (default: http://127.0.0.1:1337)

CONFIGURE IN AETHER:
  POST /api/v1/security/burp/configure
  {
    "api_url": "http://127.0.0.1:1337",
    "api_key": "your-api-key"
  }

AVAILABLE FEATURES:

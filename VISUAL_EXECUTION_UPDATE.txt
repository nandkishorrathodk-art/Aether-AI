HOW IT WORKS:
-------------

BEFORE (Background execution):
```python
# Scan ran in background - user saw nothing
result = await scanner.scan_target(target)
# 2 minutes later... results appear
```

AFTER (Visual execution):
```python
# Opens ACTUAL CMD window with scan
result = await visual_executor.run_nuclei_scan_live(target)
# User SEES scan running in real-time!
# Window stays open showing all output
```

Technical Details:
1. Uses Windows 'start' command to spawn new windows
2. Each process tracked by ID
3. /K flag keeps window open after command finishes
4. Titles set for easy identification
5. All processes manageable (can close programmatically)

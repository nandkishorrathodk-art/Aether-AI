HOW TO USE:
-----------

OPTION 1: Run Full Server
  1. Start API server:
     uvicorn src.api.main_clean:app --host 0.0.0.0 --port 8000 --reload
     
  2. In another terminal, run demo:
     python demo_live_assistant.py
     
  3. Server will be at: http://localhost:8000

OPTION 2: Direct API Calls
  curl -X POST http://localhost:8000/api/v1/live/command \
    -H "Content-Type: application/json" \
    -d '{"command": "Play Lofi hip hop on YouTube"}'

OPTION 3: WebSocket (Real-time)
  ws://localhost:8000/api/v1/live/ws/live-updates

TROUBLESHOOTING:
----------------

If server won't start:
- Check dependencies: pip install -r requirements.txt
- Check port 8000 is free: netstat -ano | findstr :8000
- Check logs in terminal

If voice doesn't work:
- edge-TTS requires internet for first download
- After first use, it works offline
- Check audio output device

If browser won't open:
- Install Playwright: playwright install chromium
- Or use existing Chrome: check browser_controller.py

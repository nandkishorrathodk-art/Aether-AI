========================================
  AETHER AI - HOW TO USE
========================================

IMPORTANT: Aether AI is an API-only system right now!
It doesn't have a chat window - you interact via:
  1. Web Browser (API Docs)
  2. Command Line Scripts
  3. API Calls

========================================
  METHOD 1: USE WEB BROWSER (EASIEST)
========================================

1. Make sure server is running (CLICK-ME.bat or RUN.bat)

2. Open browser and go to:
   http://127.0.0.1:8000/docs

3. You'll see ALL available endpoints

4. To chat:
   - Find: POST /api/v1/chat/conversation
   - Click "Try it out"
   - Edit the JSON:
     {
       "message": "Hello! Who are you?",
       "session_id": "test123",
       "stream": false
     }
   - Click "Execute"
   - See response below!

========================================
  METHOD 2: USE COMMAND LINE CHAT
========================================

1. Make sure server is running

2. Double-click: CHAT.bat

3. Type your messages and press Enter

4. Aether AI will respond!

========================================
 METHOD 3: USE OPENCLAW (WEB SCRAPING)
========================================

1. Server running? Good!

2. Open browser: http://127.0.0.1:8000/docs

3. Find: POST /api/v1/openclaw/scrape

4. Try it with:
   {
     "url": "https://example.com"
   }

5. Get scraped content!

========================================
  WHY NO CHAT WINDOW?
========================================

Aether AI is a BACKEND system (API server).
It provides services that OTHER apps use.

Think of it like:
  - Google Maps API (backend)
  - vs Google Maps App (frontend)

Aether AI = Backend (what you have)
Chat UI = Frontend (not built yet)
